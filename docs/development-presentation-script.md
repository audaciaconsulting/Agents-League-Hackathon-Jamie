# Project Development Presentation Script

## Opening

This project started with two requirements: use Copilot to help shape the project, and use an Azure cloud AI service as part of the project.

The main AI goal was to use Foundry to turn profile signals into useful game
recommendations.

## Slide 0: Setting Up The Project

Before the build-out, I set up the project with guidance from the Awesome
Copilot instruction files in this repo.

I also added a custom Copilot instruction so the AI would document every
prompt and response in the prompt log. That gave me a running record of the
development decisions behind each change.

## Slide 1: AI at the Core

The most important design choice was to keep the AI layer honest.

The app only uses public information, then sends that data to Foundry so the
model can explain patterns, summarize the profile, and suggest relevant games.

## Slide 2: Feeding Better Signals to AI

Development focused on giving the model better inputs.

The server started with Steam public profile data, then added recent play
history when available so the AI could make stronger recommendations from a
more specific signal.

## Slide 3: Making the AI Output Usable

The Angular client was built to present the AI response clearly.

That meant rendering recommendation cards, showing loading states while the
analysis ran, and fixing UI updates so the Foundry output actually appeared
when the response arrived.

## Slide 4: Making the AI Setup Reproducible

The project also added Terraform-backed Azure provisioning so the Foundry
environment could be recreated consistently.

The provisioning script reads local configuration from `.env.local`, creates
the Azure AI Services resources, and writes the connection details back for
the app to use.

## Closing

In the end, this project built a reliable AI workflow from public signals to
Foundry, and yes, the whole project, including this presentation script, was
generated by AI.

You can also find the prompt history for this presentation committed in the
repository.
